# Pixel Tower – Deploy no Railway (GRÁTIS)

## Ficheiros necessários
- `index.html` – o jogo
- `server.js` – servidor WebSocket
- `package.json` – dependências Node.js

---

## Passo a passo para publicar GRÁTIS

### 1. Cria conta no GitHub (se não tens)
→ https://github.com

### 2. Cria um repositório novo no GitHub
- Clica **"New repository"**
- Nome: `pixel-tower`
- Público ✓
- Clica **"Create repository"**

### 3. Faz upload dos 3 ficheiros
- Clica **"uploading an existing file"**
- Arrasta os 3 ficheiros: `index.html`, `server.js`, `package.json`
- Clica **"Commit changes"**

### 4. Deploy no Railway
→ https://railway.app

- Clica **"Start a New Project"**
- Escolhe **"Deploy from GitHub repo"**
- Seleciona o teu repositório `pixel-tower`
- Railway deteta automaticamente que é Node.js e faz o deploy!

### 5. Obtém o URL público
- No Railway, vai a **Settings → Networking → Generate Domain**
- Fica com um URL tipo: `https://pixel-tower-production.up.railway.app`

### 6. Joga online!
- Abre esse URL no Chrome/Edge
- Partilha o link com um amigo
- Ambos clicam em **🌐 ONLINE** → são emparelhados automaticamente!

---

## Como jogar online
1. Clica **🌐 ONLINE** no menu
2. Aparece "WAITING FOR PLAYER..." com o URL
3. Partilha o URL com um amigo
4. Quando o amigo entra e clica ONLINE, o jogo começa automaticamente!

## Controlos
- **Teclado:** SHIFT ou SPACE para largar o bloco
- **Touch/Mobile:** Toca no ecrã para largar o bloco
